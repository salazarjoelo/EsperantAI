Avisos de Terceros y Open Source

EsperantAI · Vigente desde [FECHA]

1. Alcance

EsperantAI puede incluir componentes de terceros. Cada componente se rige por su licencia correspondiente. Este documento debe distribuirse junto con el Software.

2. Componentes identificados

  • Human.js — verificar versión exacta, licencia, copyright y dependencias transitivas.
  • TensorFlow.js — verificar versión exacta y licencia Apache 2.0 si está incluido o cargado indirectamente.
  • obs-websocket-js / obs-ws — verificar versión exacta, licencia y copyright.
  • Socket.IO Client — verificar versión exacta, licencia y forma de distribución.
  • Modelos de IA Human.js — verificar licencia, fuente, versión, hashes y permiso de redistribución.

3. Obligaciones antes de producción

  • Incluir texto completo de licencias requeridas.
  • Incluir copyright notices originales.
  • Registrar versión y hash de cada archivo redistribuido.
  • Confirmar licencias de modelos, pesos y dependencias transitivas.
  • No afirmar “sólo MIT” si hay componentes Apache 2.0 u otras licencias.

4. Sin respaldo de terceros

La inclusión de nombres de terceros no implica patrocinio, afiliación o respaldo, salvo que exista acuerdo escrito.
