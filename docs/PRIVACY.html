3. Cámara, gestos y datos biométricos

EsperantAI puede procesar localmente, en la memoria del dispositivo, imágenes de cámara para detectar rostro, manos, iris, emociones o gestos mediante modelos de inteligencia artificial. Este procesamiento se realiza para convertir señales visuales en acciones configuradas por el Usuario.

Salvo que se indique expresamente lo contrario y el Usuario lo autorice, el proveedor no recolecta, almacena ni recibe video, frames, grabaciones, imágenes de cámara ni plantillas biométricas persistentes. Las señales derivadas de gestos se usan para operar la aplicación localmente.
