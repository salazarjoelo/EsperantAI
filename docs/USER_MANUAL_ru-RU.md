## Что такое EsperantAI?

EsperantAI — это **веб-приложение**, которое использует искусственный интеллект для распознавания жестов лица и рук в реальном времени и преобразует их в команды для вашего стримингового ПО. Работает с:

- **OBS Studio** 28+
- **Streamlabs Desktop**
- **vMix**
- **PRISM Live Studio**
- **XSplit Broadcaster** (бета)

А также получает события от платформ:

- **Twitch**
- **YouTube Live**
- **Kick**
- **Trovo**
- **StreamElements** (мультиплатформенный мост)

### Почему «честные жесты»?

Базовые мимические выражения и повороты головы **универсальны для всех человеческих культур** (Пол Экман, 1972). Они не лгут и не зависят от географии. EsperantAI называет такие жесты «🌐 Универсальные» и отличает их от «⚠️ Культурных» жестов (знаки руками), значение которых может различаться в зависимости от страны.

Вы сами решаете, какие жесты использовать, исходя из вашей аудитории.

## Минимальные требования

### Оборудование

- **Любая USB-веб-камера** (рекомендуется 1080p или выше)
- **CPU**: любой 4-ядерный процессор за последние 5 лет
- **RAM**: минимум 8 ГБ. Рекомендуется 16 ГБ при одновременном стриме.
- **GPU**: любой с поддержкой WebGL (подойдут даже современные встроенные)

### Программное обеспечение

- **ОС**: Windows 10/11, macOS 12+ или Linux с актуальным ядром
- **Браузер**: Chrome 90+, Edge 90+ или Firefox 100+
- **Стриминговое ПО** (хотя бы одно): OBS Studio 28+, Streamlabs Desktop, vMix, PRISM, XSplit

### Интернет

- Необходим для **активации лицензии** и каждые **7 дней** для повторной проверки
- Работает **до 7 дней офлайн** (льготный период)

## Первый запуск

### Шаг 1: Разрешите доступ к камере

При первом открытии EsperantAI браузер запросит разрешение на использование камеры. **Разрешите его**.

> Важно: EsperantAI никогда не отправляет ваше видео на сервер. Обработка выполняется на 100% локально на вашем компьютере.

### Шаг 2: Выберите камеру

Если у вас несколько камер, выберите нужную из выпадающего списка.

### Шаг 3: Проверьте распознавание

Вы увидите своё лицо на левой панели. Когда EsperantAI распознает ваше лицо, индикаторы Yaw / Pitch / Roll начнут показывать значения.

### Шаг 4: Мастер калибровки (Pro+)

Если у вас лицензия Pro или Pro+, **Мастер калибровки** запускается автоматически при первом использовании. Он измеряет ваш естественный диапазон движений и устанавливает оптимальную чувствительность. Вы можете повторно запустить его в любое время с помощью кнопки **Recalibrate**.

## Подключение стримингового ПО

### OBS Studio

1. В OBS: **Tools → WebSocket Server Settings**
2. Включите WebSocket. Запишите пароль, если вы его установили.
3. В EsperantAI: панель **Connection**
4. Стриминговое ПО: **OBS Studio**
5. WebSocket URL: `ws://127.0.0.1:4455` (по умолчанию)
6. Пароль: тот, что вы установили в OBS
7. Нажмите **Connect**

### Streamlabs Desktop

1. В Streamlabs: **Settings → Remote Control**
2. Включите Remote Control
3. Запишите API Token
4. В EsperantAI: стриминговое ПО: **Streamlabs Desktop**
5. API Token: вставьте его
6. Порт: `59650` (по умолчанию)
7. Нажмите **Connect**

### vMix

1. В vMix: **Settings → Web Controller**
2. Включите Web Controller. Порт по умолчанию: 8088.
3. В EsperantAI: стриминговое ПО: **vMix**
4. Хост: `127.0.0.1`
5. Порт: `8088`
6. Нажмите **Connect**

### PRISM Live Studio

1. PRISM Live Studio v4.0.5+ требует ручной установки плагина obs-websocket
2. Скачайте `obs-websocket` с [форума OBS](https://obsproject.com/forum/resources/obs-websocket-remote-control-of-obs-studio-made-easy.466/)
3. Скопируйте его в папку плагинов PRISM
4. Перезапустите PRISM
5. Включите WebSocket в **Tools → WebSocket Server Settings**
6. В EsperantAI: стриминговое ПО: **PRISM Live Studio** (работает так же, как OBS)

### XSplit Broadcaster (бета)

1. Установите расширение «Remote xjs» в XSplit (Settings → Extensions)
2. Включите Remote в настройках
3. В EsperantAI: стриминговое ПО: **XSplit**
4. Remote xjs Proxy URL: `ws://127.0.0.1:5555/xjs` (по умолчанию)
5. Нажмите **Connect**

> XSplit находится в **бета-версии**. Расширенные функции могут быть ограничены.

## Настройка жестов и сцен

После подключения сцены вашего ПО автоматически появятся в выпадающих списках панели **Triggers**.

### Базовое сопоставление

1. Для каждого жеста (например, «Look Left») выберите сцену из выпадающего списка
2. Когда вы выполните этот жест и удержите его стабильно ~150 мс, EsperantAI переключит сцену в вашем стриминговом ПО
3. Переключение происходит автоматически и практически мгновенно

### Мультидействие (Pro+)

С лицензией Pro или Pro+ один жест может запускать **несколько действий** одновременно:
- Переключить сцену + воспроизвести звук + показать оверлей + отправить сообщение в чат

### Включение / отключение категорий

У каждой категории есть свой флажок «Enable»:

- 🧠 **Повороты головы** (универсальные — включены по умолчанию)
- 📏 **Расстояние до лица** (приближение/отдаление)
- 👁️ **Взгляд** (движение только глазами)
- 😀 **Эмоции** (улыбка, удивление, злость, нейтральное)
- 👁️‍🗨️ **Двойное моргание**
- ✋ **Жесты руками** (культурные — отключены по умолчанию)

Отключайте ненужные категории для экономии ресурсов CPU.

## Категории жестов

### 🌐 Универсальные (одинаковое значение в любой культуре)

| Жест | Ось | Как активировать |
|---|---|---|
| Центр | — | Смотреть прямо, лицо стабильно |
| Взгляд влево | отрицательный yaw | Повернуть голову влево |
| Взгляд вправо | положительный yaw | Повернуть голову вправо |
| Взгляд вверх | отрицательный pitch | Поднять лицо |
| Взгляд вниз | положительный pitch | Опустить лицо |
| Наклон влево | отрицательный roll | Наклонить голову к левому плечу |
| Наклон вправо | положительный roll | Наклонить голову к правому плечу |
| Приближение | расстояние | Приблизить лицо к камере |
| Отдаление | расстояние | Отдалить лицо от камеры |
| Направление взгляда | взгляд | Двигать только глазами (голова в центре) |
| Улыбка | эмоция=happy | Чётко улыбнуться |
| Удивление | эмоция=surprise | Сделать удивлённое выражение |
| Злость | эмоция=angry | Сделать злое выражение |
| Нейтральное | эмоция=neutral | Расслабленное лицо |
| Двойное моргание | моргание | Быстро закрыть оба глаза дважды (< 700 мс) |

### ⚠️ Культурные (значение зависит от страны)

| Жест | Значение на Западе | Осторожность в других культурах |
|---|---|---|
| 👍 Палец вверх | Одобрение | Ближний Восток / Западная Азия: может быть оскорбительным |
| ✌️ Знак мира | Мир / победа | Великобритания / Ирландия / Австралия (ладонью внутрь): оскорбление |
| 🤘 Рога рока | Рок, метал | Италия (ладонью вниз): «корнуто» (оскорбление) |
| 👌 ОК | ОК / отлично | Бразилия / Турция / Германия: может быть оскорбительным |
| ✊ Сжатый кулак | Зависит от политического контекста | — |
| 🖐️ Открытая ладонь | «Стоп» или приветствие | Греция (моундза в чью-то сторону): сильное оскорбление |
| ☝️ Указание | Указать на что-то | Азия: указывать пальцем невежливо |

EsperantAI отмечает каждый жест соответствующим значком в интерфейсе. Выбирайте, какие из них использовать, исходя из вашей глобальной аудитории.

### 🙏 Гассё (合掌)

Особый жест: прижмите обе ладони друг к другу перед грудью (как при молитве или поклоне приветствия). Распространён в восточноазиатских культурах как знак уважения или благодарности. Распознаётся с высокой надёжностью с помощью 6 контрольных точек.

## Подключение стриминговых платформ

Чтобы EsperantAI мог получать события (пожертвования, подписки, рейды), подключите платформы, на которых вы стримите.

### Twitch

1. Создайте Client ID на https://dev.twitch.tv/console
2. Зарегистрируйте redirect URI: `https://esperantai.com/oauth-callback.html` (или ваш локальный URL)
3. В EsperantAI: панель **Platform Events** → **Twitch EventSub**
4. Вставьте ваш Client ID
5. Нажмите **Connect**
6. Откроется окно авторизации Twitch. Примите разрешения.
7. Окно закроется, и вы увидите «Twitch Connected»

### YouTube Live

1. Создайте учётные данные на https://console.cloud.google.com
2. Включите YouTube Data API v3
3. Создайте OAuth Client ID (тип: Web Application)
4. Зарегистрируйте тот же redirect URI, что и для Twitch
5. В EsperantAI: панель **Platform Events** → **YouTube Live**
6. Вставьте ваш Client ID и нажмите **Connect**

### Kick

1. Создайте приложение на https://kick.com/settings/developer
2. Зарегистрируйте redirect URI
3. В EsperantAI: панель **Platform Events** → **Kick**
4. Вставьте ваш Client ID и нажмите **Connect**
5. Kick использует OAuth 2.1 с PKCE (более безопасно)

### StreamElements (мультиплатформенный мост)

Если у вас уже есть аккаунт StreamElements, вы можете объединить Twitch + YouTube + Facebook с одним токеном:

1. Перейдите на https://streamelements.com/dashboard/account/channels
2. Скопируйте ваш JWT Token
3. В EsperantAI: панель **Platform Events** → **StreamElements**
4. Вставьте JWT и нажмите **Connect**

## Комбо: событие + жест (Продвинутый уровень)

Это магия EsperantAI: объединение **событий платформы** с **вашими жестами** в качестве подтверждения.

### Пример: поблагодарить за пожертвование жестом 👍

1. Панель **Event Triggers** → строка «💰 Donation»
2. ✅ Включите
3. Сцена: `Thank_You_Scene`
4. Требуемый жест: `👍 Thumbs up`

**Как это работает в реальном времени**:
- Поступает пожертвование → EsperantAI показывает «Waiting for gesture...»
- У вас есть 5 секунд, чтобы показать 👍
- Если вы покажете → переключается на `Thank_You_Scene` + выполняются другие настроенные действия
- Если нет → событие автоматически отменяется

### Без требуемого жеста (автоматический триггер)

Если вы оставите «Required gesture» как `— none —`, событие вызовет действие немедленно.

Полезно для:
- Автоматического переключения на праздничную сцену при рейдах
- Автоматического показа оверлея при новой подписке

## Чувствительность и мёртвая зона

### Чувствительность

Пороги определяют, насколько выраженным должен быть жест для срабатывания:

- **Yaw**: насколько сильно нужно повернуть голову вбок (по умолчанию: 0,15 рад ≈ 8,6°)
- **Pitch вверх/вниз**: вертикальный наклон
- **Roll**: боковой наклон

Увеличьте значения для более выраженных жестов. Уменьшите для большей чувствительности.

### Мёртвая зона (защита от утомления)

Если вы почти в центре (yaw < 0,05, pitch < 0,05, roll < 0,08), **НИЧЕГО не срабатывает**. Это позволяет вам двигаться естественно, не активируя триггеры микродвижениями.

### Стабильные кадры

`Stable frames` = количество последовательных кадров, в течение которых жест должен удерживаться перед срабатыванием. По умолчанию: 5 кадров (~150 мс при 30 fps).

Увеличьте, если триггеры срабатывают слишком легко. Уменьшите для более быстрой реакции.

### Перезарядка

`Cooldown (ms)` = минимальное время между переключениями сцен. По умолчанию: 500 мс.

Предотвращает «дрожание» переключателя при быстрых колебаниях.

## Устранение неполадок

### «Activation required» не исчезает после вставки лицензионного ключа

- Убедитесь, что вы скопировали ключ целиком (5 групп по 4 символа, разделённых дефисами)
- Проверьте интернет-соединение (при первой активации требуется онлайн-проверка)
- Если вы уже активировали лицензию на 3 устройствах, сначала деактивируйте одно из них
- Обратитесь в soporte@edugame.digital, если проблема сохраняется

### «Searching for face...» не исчезает, хотя ваше лицо видно

- Улучшите освещение: ваше лицо должно быть хорошо освещено
- Приблизьтесь к камере (оптимально 40–80 см)
- Закройте другие вкладки, использующие GPU (Chrome может ограничивать GPU, если их слишком много)
- Если в Chrome активен Memory Saver, отключите его для этой вкладки

### Сцены не появляются в выпадающих списках

- Убедитесь, что вы подключены к стриминговому ПО (зелёный значок «Connected»)
- Нажмите `R` для перезагрузки списка сцен
- Если по-прежнему пусто, отключитесь и подключитесь заново

### Переключение сцен происходит без ваших жестов

- Увеличьте порог yaw / pitch / roll на панели **Sensitivity**
- Увеличьте `Stable frames` с 5 до 8–10
- Убедитесь, что мёртвая зона настроена (yaw 0,05, pitch 0,05, roll 0,08)
- Проверьте, что в кадре нет других людей (несколько лиц могут вызывать нестабильность)

### Задержка распознавания

- Закройте ресурсоёмкие приложения (игры, видеомонтаж)
- Убедитесь, что используется выделенный GPU, если он есть (а не встроенный)
- Снизьте разрешение камеры, если оно 4K (1080p оптимально для распознавания)

### OBS не реагирует, хотя EsperantAI показывает «Scene changed»

- Убедитесь, что имя сцены в выпадающем списке ТОЧНО совпадает с именем в OBS (с учётом регистра)
- Убедитесь, что сцена не находится в другой коллекции сцен (Scene Collection)
- Проверьте панель **Trigger History** — если она показывает ✗ красный, есть конкретная ошибка

### Ошибка «OBS unreachable — Connect manually»

- Убедитесь, что OBS запущен
- Убедитесь, что WebSocket включён в OBS
- Если вы установили пароль в OBS, он должен совпадать точно
- Некоторые антивирусы блокируют порт 4455 — добавьте исключение
