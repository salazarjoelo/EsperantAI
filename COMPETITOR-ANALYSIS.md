## Analisis de positioning

### Lo que EsperantAI hace QUE NADIE MAS hace (verificable contra los datos arriba)
1. **Control de software de streaming via gestos faciales detectados localmente en navegador (GPU)** - ningun competidor combina (a) face tracking + (b) hand gestures + (c) triggers de OBS/Streamlabs/vMix/PRISM/XSplit. Animaze y VTube Studio solo controlan AVATARES. Plugins OBS open source son single-purpose y limitados.
2. **5 software streaming + 4 plataformas (Twitch/YouTube/Kick/Trovo) en un solo producto** - Stream Decks fisicos requieren plugin por plataforma. Mix It Up solo Twitch/YouTube/Trovo (no Kick). Ninguno hands-free.
3. **100% local sin cloud, sin trackers, sin subscription mensual** - Animaze Pro es subscription anual obligatoria. Stream Deck Mobile es subscription. Voicemod Pro tambien.
4. **13 idiomas con auto-detect** - dato no verificable en competidores; muy probablemente unicum en su mercado de habla hispana.
5. **Combo triggers (evento plataforma + gesto confirmacion)** en Pro+: no encontrado en ningun competidor.
6. **StreamElements bridge unificado Twitch+YouTube+Facebook** en Pro+: tampoco.

### Lo que SI hacen los competidores que EsperantAI debe explicar
- **Stream Decks fisicos tienen feedback tactil** - EsperantAI es software puro, no hay haptico. Mitigacion: educar que la naturalidad de un gesto facial supera el feedback de boton.
- **Animaze/VTube Studio ofrecen avatar visual** - EsperantAI no es para VTubers que quieren avatar; es para streamers de cara real o VTubers que ya usan otro software de avatar. Posicionar como complementario, no competidor.
- **Stream Decks fisicos no consumen CPU/GPU del streaming PC** - EsperantAI requiere GPU (WebGL). Mitigar con calibration y settings de FPS.

### Categoria azul/roja
**EsperantAI inaugura una categoria nueva: "Hands-Free Streaming Control via Facial AI"**. Datos verificables que lo confirman:
- No hay producto comercial dedicado en 2026 que haga control multi-software streaming via face/hand gestures.
- Los que mas se acercan son plugins open source single-purpose (Roboflow 4 gestos solo OBS) o face tracking para avatares (Animaze).
- Categoria azul, pero con riesgo de educacion de mercado (los streamers no saben que necesitan esto hasta que lo prueban).

## Recomendacion de pricing para EsperantAI (justificada)

> Esta recomendacion se basa en los benchmarks verificados arriba. Joel decide; yo solo presento el analisis.

### Para EsperantAI Pro

**Rango sugerido: $39 - $59 USD one-time**

Justificacion verificable:
- **Floor $39:** Voicemod lifetime $39.99 [MENCIONADO 2026]. Posicionar Pro como "equivalente en valor a Voicemod Pro lifetime, pero para control hands-free".
- **Mid $49:** Stream Deck Mobile lifetime exact $49.99 [MENCIONADO Elgato 2026]. EsperantAI Pro reemplaza un Stream Deck Mobile PRO + hace mas (face + gestos vs touch en pantalla).
- **Ceiling $59:** Stream Deck Mini hardware $59.99 [VERIFICADO Elgato 2026]. EsperantAI Pro vs un Stream Deck Mini: mas triggers (18 vs 6), software universal multi-OBS, hands-free. Joel puede argumentar paridad de valor.
- **Por encima de $59 hay friccion:** Touch Portal Pro $13.99 [VERIFICADO] crea anclaje psicologico de "software puro = barato". Subir mucho requiere educacion fuerte.

### Para EsperantAI Pro+

**Rango sugerido: $79 - $99 USD one-time**

Justificacion verificable:
- Combo triggers + StreamElements bridge + triggers ilimitados son features verificablemente sin competidor directo.
- **Floor $79:** ~30% premium sobre Pro $59, consistente con uplift Plus->Pro tipico (Animaze Plus $19.99 -> Pro $99.99 = 5x; mas agresivo de lo que recomiendo).
- **Mid $89:** Stream Deck Pedal $89.99 [VERIFICADO]. Joel puede argumentar "reemplaza un pedal y mas".
- **Ceiling $99:** Animaze Pro anual $99.99 (subscription) [VERIFICADO]. EsperantAI Pro+ one-time al mismo precio que un anio de Animaze Pro = ventaja clara comunicable. Tambien por debajo de Animaze Pro 22 Steam lifetime $129.99 [MENCIONADO].

### Modelo

- **One-time fuertemente recomendado** (alineado con vision Joel + preferencia mercado hispano + diferenciacion vs Animaze/Stream Deck Mobile subscription).
- Posible early-bird: Pro $29 / Pro+ $59 primeros 100 (lifetime, sin recurring) para validacion + reviews + caso de uso.
- Pricing localizado LATAM (descuento PPP ~40-50% si Joel quiere abrir mercado hispano sin presion). **No tengo dato cuantitativo del descuento exacto recomendable.**
